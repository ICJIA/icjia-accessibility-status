## Key Features

### Public Dashboard
View real-time accessibility scores for all monitored websites with interactive charts and detailed site information.

### Detailed Site Views
Interactive charts showing historical score progression with Axe and Lighthouse accessibility metrics.

### Score Improvement Graphs
Visual trend charts for both Axe and Lighthouse scores showing site-by-site improvements over time.

### Multi-Format Exports
Export data in JSON, CSV, and Markdown formats for reporting and analysis.

### Secure Admin Interface
Protected authentication system with 15-day persistent sessions and role-based access control.

### API-Only Data Ingestion
Secure REST API for programmatic data uploads with API key authentication.

### Activity Log
Track all significant events including site creation, updates, and API imports.

- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS
- **Backend**: Express, Node.js, TypeScript
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Cookie-based sessions with bcrypt hashing
